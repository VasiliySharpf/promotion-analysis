## Promotion analysis service

### Используемые технологии:
* Spring Boot (ver. 3.1.2)
* Spring Web
* Spring Data JPA (Hibernate)
* PostgreSQL
* Maven
* Liquibase
* Validation (Hibernate Validator)
* Open API (Swagger)
* Lombok
* Spring Test (JUnit 5, Mockito)
* Testcontainers
* Docker + Docker Compose
